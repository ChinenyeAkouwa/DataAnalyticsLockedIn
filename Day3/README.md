### Gaps Identified:

- No measurable KPIs defined at the start.
- No evaluation stage to validate whether recommendations would solve the problem.

### Reflection:
This exercise showed me that strong analytics requires more than clean data and good visuals. It requires intentional structure from problem definition to measurable impact. Going forward, I will incorporate clearer success metrics and deeper root cause analysis into my projects.
