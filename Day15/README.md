Product Demand Analytics – Excel Case Study

Overview

This project analyzes two years (2018–2019) of order data (50,000+ rows) from a multi-warehouse distribution model.

Dataset Fields

• Product category 

• Product  (SKU)

• Warehouse locations

• Customer ZIP codes

• Order Date

• Order Time

• Order Demand (units)

• Transaction values

Objectives

Evaluate revenue volatility across time
Identify seasonal demand patterns
Assess product portfolio performance
Analyze warehouse and geographic distribution efficiency
Explore relationships between order size and transaction value

Approach

Data cleaning and preparation
Time intelligence structuring (Year, Month, Quarter, Day)
KPI definition and calculation
Trend and variance analysis
Executive-level dashboard development

Outcome

The final deliverables will include:
An executive dashboard
Supporting analysis sheets
A concise business insight summary with recommendations
