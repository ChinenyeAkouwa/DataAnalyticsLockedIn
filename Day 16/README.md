Project: Product Demand Analytics – B2B Software Company

Phase Completed: Data Cleaning

Overview:
This project simulates a BI Analyst workflow analyzing two years of order data for a B2B software company. The goal is to understand demand trends and prepare the data for executive-level insights.

Data Cleaning Steps:

Checked dataset for blanks, duplicates, and inconsistencies

Calculated Q1, Q3, and IQR to identify outliers in Order Demand

Flagged and adjusted 7,700 extreme values, keeping 42,299 as valid

Added Adjusted Order Demand column to reflect realistic business patterns

Verified that no lower-bound adjustments were needed — demand can’t be negative

Purpose:

Ensure accurate KPIs and reliable trends

Provide a solid foundation for pivot tables, dashboards, and insights

Next Steps:

Calculate KPIs (Revenue, Average Order Value, Units per Order)

Build Month-Year pivot tables and analyze MoM growth

Create executive dashboard and insight summary

Tools: Excel, Pivot Tables, Formulas, Outlier Analysis
